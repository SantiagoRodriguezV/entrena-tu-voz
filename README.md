# Entrena tu Voz

Prototipo académico de aplicación móvil para aprendizaje vocal (Expo 57 + React Native + TypeScript).

## Inicio rápido

```bash
npm install
npx expo start          # Web (micrófono simulado)
npx expo run:android    # Android con micrófono real
```
